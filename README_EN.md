# Beauty Light Web Application

[中文](README.md)

✨ A web-based real-time beauty lighting application that helps you look your best anytime, anywhere!

## Features

- 🎥 Real-time video preview
- 🎨 Rich filter settings
  - Brightness adjustment (0-200%)
  - Saturation adjustment (0-200%)
  - Hue adjustment (0-360°)
- 🎭 Preset filter styles
  - Normal mode
  - Warm tone
  - Cool tone
  - Vintage style
  - Dramatic style
- 💫 Real-time effect preview
- 📱 Responsive design, supporting various devices

## Usage Instructions

1. When opening the application, your browser will request camera permission, please allow access
2. Adjust the following parameters using the sliders:
   - Brightness: Adjust the overall brightness of the image
   - Saturation: Adjust the color intensity
   - Hue: Adjust the overall color tone
3. Click preset filter buttons to quickly apply different style effects

## Technical Implementation

- Frontend Technologies:
  - HTML5 (Video stream processing)
  - CSS3 (Modern UI design)
  - JavaScript (Real-time filter processing)
  - Font Awesome 6.4.0 (Icon support)
  - Google Fonts - Poppins (Font support)

## Development Setup

1. Clone the project locally
2. Open `index.html` using a modern web standards-compliant browser
3. Ensure your camera is working properly and grant permissions

## Notes

- Requires a modern browser with WebRTC support
- Browser camera access permission is required
- Recommended to use in well-lit environments for best results
