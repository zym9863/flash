# 美颜补光灯 Web 应用

[English](README_EN.md)

✨ 一个基于网页的实时美颜补光应用，让你随时随地展现最佳状态！

## 功能特点

- 🎥 实时视频预览
- 🎨 丰富的滤镜设置
  - 亮度调节（0-200%）
  - 饱和度调节（0-200%）
  - 色相调节（0-360°）
- 🎭 预设滤镜风格
  - 正常模式
  - 暖色风格
  - 冷色风格
  - 复古风格
  - 戏剧风格
- 💫 实时预览效果
- 📱 响应式设计，支持各种设备

## 使用说明

1. 打开应用后，浏览器会请求摄像头权限，请允许访问
2. 通过滑动条调节以下参数：
   - 亮度：调节画面整体明暗度
   - 饱和度：调节画面色彩的鲜艳程度
   - 色相：调节画面的整体色调
3. 点击预设滤镜按钮，快速应用不同风格效果

## 技术实现

- 前端技术：
  - HTML5 (视频流处理)
  - CSS3 (现代化UI设计)
  - JavaScript (实时滤镜处理)
  - Font Awesome 6.4.0 (图标支持)
  - Google Fonts - Poppins (字体支持)

## 开发环境设置

1. 克隆项目到本地
2. 使用支持现代Web标准的浏览器打开 `index.html`
3. 确保摄像头正常工作并授予权限

## 注意事项

- 需要使用支持WebRTC的现代浏览器
- 需要允许浏览器访问摄像头
- 建议在光线充足的环境下使用，以获得最佳效果
